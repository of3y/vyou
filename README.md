# VYou

Photo-first community weather app. Each report is a view-cone on a shared map, classified and reconciled against satellite + radar by Claude Managed Agents on Opus 4.7.

Submission for the [Built with Opus 4.7 Claude Code hackathon](https://cerebralvalley.ai/e/built-with-4-7-hackathon) (Anthropic × Cerebral Valley, 2026-04-21 → 2026-04-26).

Status: Day 1 scaffold. Details land here as the build progresses.
