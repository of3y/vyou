# severe-weather-reporting

Reference knowledge for the agents that classify and reconcile field-captured weather photos. The skill names the phenomena VYou recognizes, the visual features diagnostic of each, the situations that warrant elevated report priority, and the rubrics for judging report quality. The Classifier agent cites entries here when naming a phenomenon; the Reconciliation agent cites them when judging consistency between the photo and the radar/satellite context; the Deep Researcher cites them when drafting personalized briefings; the Classifier Validator uses the same taxonomy as ground truth for the supervised benchmark.
