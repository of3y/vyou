# demo/

Remotion project for the VYou three-minute submission video.

## Quick start

```bash
cd demo
npm install
npm run start          # opens the Remotion studio in a browser
npm run render         # writes an MP4 into renders/ with a dated filename
```

## Structure

- `src/Root.tsx` registers the composition.
- `src/HelloVYou.tsx` is the Day 1 hello-world — replace piece by piece as the beat sheet in `docs/demo-script.md` gets built out.
- `renders/` holds committed final MP4 artifacts. Use the dated draft filenames for iterations; name the final `final-2026-04-26.mp4`.

## Discipline

First render by end of Day 2 (hello-world is fine). First full beat sheet render by Thursday evening. Final render Saturday afternoon Munich time with Sunday as buffer. See `docs/demo-script.md` for the beat-by-beat plan.
